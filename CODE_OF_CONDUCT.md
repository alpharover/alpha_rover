# Code of Conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/) Code of Conduct, version 2.1.

## Our Pledge
We pledge to make participation in this project and our community a harassment‑free experience for everyone.

## Our Standards
Be respectful. No harassment, hate speech, or personal attacks. Assume good intent; critique ideas, not people.

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Scope
This Code applies within project spaces and in public spaces when an individual is representing the project.
